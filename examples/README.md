# Eventra Examples

Examples for using Eventra in different JavaScript environments.

## Available Examples

- Browser
- Node.js
- React
- Next.js
- Next.js App Router
- Vue
- Angular
- Svelte
- NestJS
- Express
- Serverless
- TypeScript

## Install

### npm

```bash
npm install @eventra_dev/eventra-sdk
```

### pnpm

```bash
pnpm add @eventra_dev/eventra-sdk
```

### yarn

```bash
yarn add @eventra_dev/eventra-sdk
```
